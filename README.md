# Vue脚手架
## ref属性
1. 用来给元素或者子组件注册引用信息(id的替代者)
2. 应用在*html*标签上获取的是真实的DOM元素，应用在*组件标签*上是组件实例对象(vc)
3. 使用方式
    * 打标识: ```<h1 ref="xxx">....</h1>``` 或者 ```<School ref="xxx"></School>```
    * 获取: ```this.$refs.xxx```
